Goal

Fill in a 9×9 grid with digits so that each column, row, and 3×3 section contain the numbers between 1 to 9.

Play

In a new game start, the 9×9 grid will have some of the squares filled in. You can use logic to fill in the missing digits and complete the grid.

Don’t forget, error if bellow:

  1. Any row contains more than one of the same number
  2. Any column contains more than one of the same number
  3. Any 3×3 grid contains more than one of the same number
